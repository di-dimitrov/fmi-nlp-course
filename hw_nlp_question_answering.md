# Homework assignment #2

## Name: Automatic question answering

Participate in **ImageCLEF MultimodalReason** competition. Follow the instructions on the official task website ([https://www.imageclef.org/2026/multimodalreasoning](https://mbzuai-nlp.github.io/ImageCLEF-MultimodalReasoning/2026)):

* Register for the task - follow the instructions and select ImageCLEF lab. You are free to select others if you want to participate separately from the homework.
* Choose a task you want to participate in: OpenQA.
* Download the available data. There is already a dataset on this topic from the same authors. If you want to use it, find it.
* Decide on approach(es) to try out. The data can be used as text-only or with an image. You can use any of the provided details in the dataset. 
* Submit your results in the official leaderboard.
* Write short description of the approach(es). The description should contain explanation of the approach you have used and steps taken to process the data. Furthermore, the results must be reported with a short discussion, especially if more than 1 approach was tested.

**Deadline:** 8th of June

**Teams are not allowed!**

**Submission format:** submit .zip of the used code and a short (up to one page) description of your system. The description should only focus on what you have done, without unnecesary details like data stats. Report your score!
